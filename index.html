Viral Recombination Analysis

A 3D visualization tool built with Streamlit and Plotly to analyze viral genetic sequences using Topological Data Analysis. This project, part of my Master's thesis, detects recombination events in viruses by transforming genomic distance matrices into simplicial complexes, revealing evolutionary patterns like reassortment in H7N9 strains.
