Projects / Proyectos

Breast Cancer Subtype Classification / Clasificación de Subtipos de Cáncer de Mama

Analyzed The Cancer Genome Atlas (TCGA) gene expression data to classify Breast Invasive Carcinoma subtypes. This project utilizes a machine learning workflow in Python (Pandas, Scikit-learn) involving data preprocessing, feature selection, SMOTE for handling class imbalance, and a RandomForestClassifier model to predict cancer subtypes from mRNA profiles.

Se analizaron datos de expresión génica de The Cancer Genome Atlas (TCGA) para clasificar subtipos de Carcinoma Invasivo de Mama. Este proyecto utiliza un flujo de trabajo de machine learning en Python (Pandas, Scikit-learn) que incluye preprocesamiento de datos, selección de características, SMOTE para manejar el desbalance de clases y un modelo RandomForestClassifier para predecir subtipos de cáncer a partir de perfiles de ARNm.

3D Visualization of TDA Applied to Genomic Sequences / Visualización 3D de TDA Aplicada a Secuencias Genómicas

A 3D visualization tool (Streamlit/Plotly) analyzing the behavior of a specific genetic sequence under study (recombinant Influenza H7N9) by applying Topological Data Analysis (TDA) to detect horizontal gene transfer. Part of my Master's thesis.

Herramienta de visualización 3D (Streamlit/Plotly) que analiza el comportamiento de una secuencia genética de estudio (recombinante Influenza H7N9) al aplicarle Análisis Topológico de Datos (TDA) para detectar transferencia génica horizontal. Parte de mi tesis de Maestría.
